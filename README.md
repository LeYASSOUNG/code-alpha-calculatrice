# TASK 2 : Calculatrice Avancée 🧮

Une calculatrice web élégante utilisant l'esthétique Glassmorphism, intégrant des fonctions basiques et scientifiques.

## ✨ Fonctionnalités
- **Opérations Mathématiques :** Additions, soustractions, multiplications, divisions, avec respect de la priorité des opérations via la fonction `eval()` sécurisée.
- **Mode Scientifique (🔬) :** Panneau extensible incluant des fonctions avancées telles que `sin`, `cos`, `tan`, `log`, `ln`, `√`, `^` et `π`.
- **Historique Persistant :** Enregistrement des 20 derniers calculs dans le `LocalStorage`.
- **Support Clavier :** Contrôlez entièrement la calculatrice à l'aide de votre pavé numérique (Entrée pour '=', Échap ou Retour arrière pour supprimer).
- **Animations (UI/UX) :** Effet de lueur (Néon Glow) sur l'écran et ondes lumineuses (Ripple Effect) au clic des boutons.

## 🛠️ Lancement
Ouvrez le fichier `calculatrice.html` dans votre navigateur web.
